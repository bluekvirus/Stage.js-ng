#WebUI Skeleton
Blueprint for single page web/hybrid front-end projects.

##Usage
###1. Create your project:
```
git clone https://github.com/bluekvirus/skeleton-webui.git .
```

###2. Initialize it:
```
bower install

cd ./build
npm install
```

###3. Build:
```
cd ./build
gulp [help]
```
See Gulpfile under `./build`.
